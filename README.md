# Groupie Tracker

Projet web réalisé en Go dans le cadre du projet **Groupie Tracker**.

L’application consomme l’API Groupie Tracker et permet d’afficher une liste d’artistes, leurs détails et de gérer des favoris persistants.

---
Accès à l’application :
http://localhost:8080/films


## Technologies
- Go
- HTML / CSS
- API Groupie Tracker

---

## Installation et lancement

### Prérequis
- Go installé (version 1.21 ou compatible)

### Lancer le projet
À la racine du projet :

```bash
go mod tidy
go run .
